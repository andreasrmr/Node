Om jQuery

jQuery er et javascript bibliotek som gør det nemmere og skrive javascript. jQuery API Dokumentation findes her

Basic syntax:

$(selector).action

  • Et $ til at definere jQuery
  • selector som bruges til at finde et specifik html element.
  • action er hvad der skal gøres ved det valgte html element

Hele listen med selectors kan findes her

jQuery eksempel:

1. Ændre html tekst for div med specifik id.

        <html>

            <div id="content"></div>

            <script> 

                $("#content").html("<h1>Hello world</h1>"); 

            </script> 

        </html>
