Oprettelse af nyt projekt (express framework)

Dependencies og NPM

  1. Opret ny mappe og kald den hvad du vil
  2. Opret en fil i mappen med navnet: package.json
  3. package.json skal indeholde et json object og derfor skal det som minimum indeholde: {}
  4. Dependencies kan defineres i package.json
  5. I snippet er der defineret 2 dependencies
  6. Find versionsnummer, flere pakker og information herom på: npmjs.com
  7. NPM står for Node Package Manager
  8. Når ens package.json er installaret skrives i terminalen i roden af projektet: npm install
  9. Nu hentes pakkerne til en ny mappe: node_modules
